# Valentine's Day Interactive Project

## Overview
This project is designed to create an interactive experience to celebrate Valentine's Day. Users can engage with various features that allow them to celebrate love and friendship through fun interactions.

## Features
- Interactive games
- Love quotes generator
- Customizable greeting cards
- User-friendly interface

## Setup Instructions
To set up the project locally, please follow these instructions:

1. **Clone the repository**:  
   Run the following command to clone the repository to your local machine:  
   ```bash
   git clone https://github.com/ringgasbriliaan-wq/untukunatersayang.git
   ```  

2. **Navigate to the project directory**:  
   ```bash
   cd untukunatersayang
   ```  

3. **Install dependencies**:  
   Make sure you have `npm` installed. Then run:  
   ```bash
   npm install
   ```  

4. **Run the project**:  
   To start the interactive project, use the following command:  
   ```bash
   npm start
   ```  

5. **Open your browser**:  
   Go to `http://localhost:3000` to view the project in your browser.

## Customization Guide
To customize the project, you can modify the following files:
- **Game settings**: Update the settings in `src/settings.js` for different game modes.
- **Greeting card templates**: Edit the card designs in the `src/cards` directory.
- **Love quotes**: Add or change love quotes in the `src/quotes.json` file.
